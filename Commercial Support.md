# Commercial Support

Many companies rely on `zxing-cpp` to power barcode scanning and generation in commercial products, industrial systems, mobile apps, medical devices, logistics solutions, and embedded hardware.

If `zxing-cpp` is an important part of your product, professional support is available directly from the project's maintainer.

## Services

### Priority Technical Support

Receive help directly from the maintainer for topics such as:

- Integration into existing software
- Performance optimization
- Difficult barcode recognition problems
- Platform-specific issues
- Build and packaging problems
- API migration

Support is provided via email and, where appropriate, video calls.

---

### Consulting and Custom Development

Need something beyond standard support?

Examples include:

- Adding support for new barcode features or variants
- Performance tuning for high-throughput applications
- Embedded and resource-constrained systems
- Mobile camera integration
- Custom decoding workflows
- Code reviews
- Architecture advice

Both hourly consulting and fixed-price projects are possible.

Whenever possible, improvements are contributed back to the open-source project for the benefit of the entire community.

## Why work directly with the maintainer?

`zxing-cpp` has been actively developed for many years and is used in products across a wide range of industries.

Working directly with the maintainer provides:

- Deep knowledge of the decoding and encoding algorithms
- Faster diagnosis of difficult issues
- Guidance on best practices and API usage
- Reduced integration risk
- Direct influence on future development
